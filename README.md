# 100 Days of Machine Learning

This repository accompanies CampusX's **"100 Days of Machine Learning"** YouTube series, a structured, beginner-to-advanced ML curriculum delivered in Hindi. Each folder corresponds to a day in the series and contains the code, notebooks, and datasets used in that session.

## What's Covered

**Data Handling & Sourcing**
- Reading data from CSV, JSON, and SQL
- Fetching data via APIs
- Building DataFrames through web scraping

**Exploratory Data Analysis**
- Descriptive statistics
- Univariate and bivariate analysis
- Automated EDA with Pandas Profiling

**Feature Engineering**
- Scaling (standardization, normalization)
- Encoding (ordinal, one-hot)
- Column Transformer and Sklearn Pipelines
- Function and Power Transformers
- Binning and binarization
- Handling mixed variables, dates, and times

**Missing Data**
- Complete case analysis
- Numerical and categorical imputation
- Missing indicators, KNN Imputer, Iterative Imputer

**Outliers & Dimensionality Reduction**
- Outlier detection using Z-score, IQR, and percentiles
- Feature construction and splitting
- Principal Component Analysis (PCA)

**Regression**
- Simple and multiple linear regression
- Gradient descent (and its variants)
- Polynomial regression
- Regularization: Ridge, Lasso, ElasticNet

**Classification**
- Logistic regression
- Classification metrics

**Ensemble Methods & Clustering**
- Random Forest, AdaBoost, Gradient Boosting
- Stacking and Blending
- K-Means clustering

## About This Repo

The repo is designed as a practical, code-first companion to the video series, letting learners follow along and reproduce every concept themselves. It is free, community-supported (2.7k+ stars, 2.5k+ forks), and remains one of the most widely used Hindi-language ML learning resources on GitHub.

> **Note:** This description is based on the current folder structure (through ensembles and clustering). Deep learning and deployment modules mentioned in the course overview were not found as dedicated folders in the repo at the time of writing — worth confirming against the live YouTube playlist before using this description officially.
